# ShowStrong / Training Protocol — Android releases

Signed release APKs for the **ShowStrong** Android app (`uk.simplyautomation.training`),
published here so they can **auto-update via [Obtainium](https://github.com/ImranR98/Obtainium)** —
no Play Store needed.

## Install / auto-update (one-time setup)
1. Install **Obtainium** (from F-Droid or its GitHub releases).
2. In Obtainium, tap **Add App** and paste this repo URL:
   `https://github.com/wdanielz/training-protocol-releases`
3. Install the latest release. From then on, Obtainium checks here and offers a
   one-tap update whenever a new version is published.

All builds are signed with the same key (CN=Training Protocol), so updates install in place.
This repo holds **only the APKs** — no source, no signing key.
